Smart Delivery and Route Optimization System - Project Progress
Project Overview

This is a Dockerized microservices-based backend system for a Smart Delivery and Route Optimization platform. The architecture follows distributed system principles using independent services connected through an API Gateway. The platform is being developed as a hackathon MVP using free/open-source technologies and MongoDB Atlas as the cloud database.

9. Bug Fix — API Gateway POST Body Forwarding ✅
Issue Found
All POST/PATCH requests through the API Gateway were hanging indefinitely.
Root Cause
express.json() was registered globally in the gateway before proxy routes.
Express consumed the request body stream, leaving nothing for http-proxy-middleware to forward.
The proxy sent an empty body and waited forever for a response.
Fix Applied
Removed express.json() from global middleware in api-gateway/src/server.js
Added if (req.body) re-stream guards in each onProxyReq handler
Result
All proxy POST/PATCH requests now work correctly end-to-end.
